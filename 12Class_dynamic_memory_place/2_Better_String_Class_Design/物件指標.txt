之前是在類別(class)的建構子，使用new與delete。

Class_Name Object_Name;  (String name;)

現在我們在物件(Object)用 new與delete。:
--------------------------------------------------------
String * favorite = new String(saying[choice]);
....
delete favorite;
--------------------------------------------------------

String * favorite = new String(saying[choice]);
不只配置名稱字串的空間，還包括物件，也就是說包括儲存字串位址的str指標和len "成員的空間"
(但不會配置static 變數的空間。)
而且在產生物件的時候會呼叫 建構函數。

delete favorite;
當不需要此物件時，程式利用delete刪除。
